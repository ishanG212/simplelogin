WHAT IS THIS?
=============
Simplelogin Module provides plain structure for 
user login, user register and user password pages.
And Customize the login screens background Image with easy steps.

INSTALLATION
============
Install the Simplelogin module the usual way at admin/modules.

CONFIGURATION
=============
Configure the background image goto admin/config/simplelogin/loginbg.
